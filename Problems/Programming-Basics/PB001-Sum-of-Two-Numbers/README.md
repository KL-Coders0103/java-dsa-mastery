# Q001 - Sum of Two Numbers

## Difficulty
⭐ Easy

## Topic
Programming Basics

## Platform
Custom

---

## Problem Statement

Write a Java program that reads two integers and prints their sum.

---

## Input Format

The first line contains an integer A.

The second line contains an integer B.

---

## Output Format

Print the sum of A and B.

---

## Constraints

-10^9 <= A,B <= 10^9

---

## Example

### Input

10
20

### Output

30

---

# Thinking

## Inputs

Two integers

---

## Output

Sum of two integers

---

## What is the question asking?

Read two numbers and print their sum.

---

## Brute Force Idea

Read both numbers.

Add them.

Print the result.

---

## Better Idea

No optimization is needed because only one addition operation is performed.

---

## My Thoughts

This problem helps understand input, output, variables, and arithmetic operations in Java.

# Algorithm

Step 1

Read first integer.

Step 2

Read second integer.

Step 3

Add both numbers.

Step 4

Store the result.

Step 5

Print the result.

Input

A = 15

---

Step 1

Step 2

---

# Complexity Analysis

Time Complexity

O(1)

Reason

Only one addition operation.

---

Space Complexity

O(1)

Reason

Only one variable is used to store the result.
